bigdata_analysis
==
这个repo 是本人实践过的数据分析项目集合,每个项目都会包含一个友好的说明文档，用来阐述和展示整个开发流程，同时也会提供相关的数据集，以供下载练习。<br/>

tip
==
* 项目主要使用的编程语言是 python、sql、hql<br/>
* .ipynb 可以用 jupyter notebook 打开，如何安装, 可以参考[jupyter notebook](http://blog.turboway.top/article/jupyter/)<br/>
